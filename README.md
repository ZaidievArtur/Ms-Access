Торгово-закупочная фирма. 

Имеется список наименований товаров для потребителей (например, лампа 100W, уголок 3дюйма, труба 10 мм…), список поставщиков товаров с указанием, что именно они продают (тип товара, цена, оптовая скидка).
Склад – наполняется по нашим заказам от поставщиков; цены поставщиков могут изменяться. 
Стартуем с некоей суммы, которая у нас уже есть, и склада, который уже чем-то набит.
Требуется поддержка:
	Оформления заказа от потребителя;
	Автоматич. генерации нашего заказа для поставщика, если у нас не хватает товара для потребителя;
	Просто оформления нашего заказа у поставщика, если есть средства.
	Возврата товара потребителем с возмещением полной стоимости по некоторым видам товаров (по другим видам возврат не возможен, например, по материалам)
	Отчеты: текущее состояние склада; перечень удовлетворенных заказов за произвольный период (заказы не отвергаем вообще); наша текущая финансовая картина (наличка + стоимость товара на складе по группам); фин. отчет по операциям за год.
